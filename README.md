# Claudito Releases

Public releases and installers for Claudito.

This repository does **not** contain source code. The source code repository is private.

## Windows Install

1. Download `claudito-external-v2.1.0-windows.zip` from the latest release.
2. Extract the zip.
3. Open PowerShell in the extracted folder.
4. Run:
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   .\install.ps1
   ```
5. After installation, run `clt`.
